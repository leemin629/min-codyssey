50대 직장인 이희망 페르소나를 기반으로, 신앙과 현실의 괴리감을 줄여주는 콘텐츠 또는 AI 코칭 서비스 방향을 정한다.

\## 2. 회의 시작 분위기

회의는 조용하게 시작되었다.

첫 화면에 띄워진 문장은 이것이었다.

\> “세상에서 잘 사는 것과 천국에 가는 삶은 왜 이렇게 다르게 느껴질까?”

잠시 침묵이 흘렀다.

누구도 바로 말을 꺼내지 못했다.

기획자는 말했다.

\> “이희망 부장님은 단순히 피곤한 사람이 아닙니다.

\> 그는 인생의 방향을 묻고 있는 사람입니다.”

이 한마디로 회의의 방향이 정해졌다.

\## 4. 주요 논의 내용

\### 1) 이희망의 진짜 문제는 무엇인가?

사용자 분석 담당자는 이희망의 문제를 이렇게 정리했다.

| 겉으로 보이는 문제 | 실제 내면의 문제 |

|---|---|

| 일이 많다 | 삶의 방향이 불안하다 |

| 시간이 없다 | 하나님과 멀어진 느낌이 든다 |

| 피곤하다 | 지금의 삶이 천국을 향하는지 확신이 없다 |

| 책임이 무겁다 | 신앙과 현실 사이에서 갈등한다 |

회의에서는 이희망의 핵심 고민을 다음 문장으로 정리했다.

\> “나는 열심히 살고 있는데, 과연 바르게 살고 있는가?”

\---

\### 2) 서비스가 주면 안 되는 메시지

참석자들은 이희망에게 단순한 종교적 훈계는 도움이 되지 않는다고 보았다.

피해야 할 메시지는 다음과 같다.

| 피해야 할 메시지 | 이유 |

|---|---|

| “더 기도하세요.” | 너무 뻔하고 부담을 줄 수 있음 |

| “세상을 내려놓으세요.” | 직장 현실을 무시하는 말처럼 들릴 수 있음 |

| “믿음이 부족해서 그렇습니다.” | 죄책감만 키울 수 있음 |

콘텐츠 담당자는 말했다.

\> “이희망 부장님에게 필요한 건 혼내는 말이 아니라,

\> 출근길에 붙잡을 수 있는 한 문장입니다.”

\---

\### 3) 서비스가 줘야 하는 메시지

회의 결과, 이희망에게 필요한 메시지는 다음 방향으로 정리되었다.

| 메시지 방향 | 설명 |

|---|---|

| 공감 | 50대 직장인의 무게를 인정한다 |

| 관점 전환 | 직장도 신앙을 실천하는 장소로 바라본다 |

| 작은 실천 | 거창한 변화보다 하루 5분 행동을 제안한다 |

| 평안 회복 | 천국은 미래만이 아니라 지금의 삶에서도 시작됨을 알려준다 |

핵심 문장은 다음과 같이 결정되었다.

\> “천국을 향한 삶은 현실을 버리는 것이 아니라,

\> 현실 속 선택을 하나님께 가까이 맞춰가는 것입니다.”

\## 5. 서비스 콘셉트 제안

회의 끝에 나온 서비스 이름 후보는 다음과 같다.

| 후보 이름 | 의미 |

|---|---|

| 오늘의 천국 연습 | 하루 속에서 천국의 방향을 연습한다 |

최종적으로 가장 좋은 반응을 얻은 이름은 다음이었다.

\> \*\*오늘의 천국 연습\*\*

이유는 간단했다.

천국을 먼 미래의 장소가 아니라, 오늘의 삶에서 연습하는 방향으로 보여주기 때문이다.

**Claude Fable 5**

**회의록: 이희망 부장을 위한 AI 코칭 서비스**

**회의명:** 페르소나 기반 신앙·직장 균형 코칭 서비스 기획 회의**  
작성자:** Claude**  
작성 원칙:** 한 문장이라도 군더더기 없이, 본질만 남긴다

1\. 문제의 본질 (한 단락 요약)

이희망 부장은 실패한 사람이 아니다. 그는 **두 가지 성공을 동시에 원하는 사람**이다. 직장에서는 인정받고 싶고, 신앙에서는 부끄럽지 않고 싶다. 문제는 세상이 이 둘을 '양자택일'처럼 보이게 만든다는 것이다. 따라서 이 서비스가 답해야 할 질문은 단 하나다. **"둘 다 지키는 삶은 가능한가?"**

2\. 갈등 구조 분석

이희망 님의 하루에서 갈등이 발생하는 지점은 세 곳으로 압축된다.

|     |     |     |     |
| --- | --- | --- | --- |
| **갈등 지점** | **세상의 요구** | **신앙의 요구** | **충돌 순간** |
| **성과 압박** | 수단보다 결과 | 결과보다 정직 | 무리한 실적 지시 |
| **관계 경쟁** | 동료는 경쟁자 | 이웃은 사랑 대상 | 후배와의 승진 경쟁 |
| **시간 배분** | 야근과 회식 | 예배와 묵상 | 수요일 저녁 회의 |

�� **핵심 통찰:** 갈등은 '큰 사건'이 아니라 \*\*'작은 선택의 순간'\*\*마다 반복된다. 그러므로 서비스도 큰 결단이 아닌, 작은 선택을 도와야 한다.

3\. 서비스 방향 결정

세 가지 방향을 검토한 뒤, 하나를 선택했다.

|     |     |     |     |
| --- | --- | --- | --- |
| **방향** | **내용** | **판단** | **이유** |
| A안  | 신앙 지식 제공형 | 기각  | 그는 몰라서가 아니라 지쳐서 흔들린다 |
| B안  | 감정 위로 중심형 | 기각  | 위로만으로는 내일 아침이 달라지지 않는다 |
| **C안** | **선택 동반형** | **채택** | 갈등의 순간마다 곁에서 함께 결정한다 |

**채택된 C안의 정의:**

정답을 주는 AI가 아니라, 이희망 님이 흔들리는 순간에 \*\*"지금 이 선택, 함께 생각해봅시다"\*\*라고 말을 거는 AI

4\. 핵심 기능 설계

C안을 구현하는 기능은 세 개면 충분하다. 많은 기능은 오히려 그를 더 피곤하게 만든다.

|     |     |     |
| --- | --- | --- |
