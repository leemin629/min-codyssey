## 3. 핵심 페르소나 요약

| 구분 | 내용 |
|---|---|
| **이름** | 이희망 |
| **나이** | 50세 |
| **직업** | 중소기업 부장 |
| **현재 상황** | 치열한 경쟁 속에서 쫓기듯 살아가며, 직장과 가정의 책임을 동시에 감당하고 있음 |

| **니즈** | 신앙과 삶의 괴리감을 줄이고, 일상 속에서 진정한 평안과 행복을 찾고 싶음 |

## 4. 문제 정의

| 겉으로 보이는 문제 | 실제 내면의 문제 |
|---|---|
| 일이 많고 늘 바쁨 | 삶의 방향이 맞는지 불안함 |
| 기도와 묵상 시간이 부족함 | 하나님과 멀어진 느낌이 듦 |
| 육체적으로 피곤함 | 지금의 삶이 천국을 향하는지 확신이 없음 |
| 직장과 가정의 책임이 무거움 | 신앙과 현실 사이에서 갈등함 |

### 핵심 질문

> **"나는 열심히 살고 있는데, 과연 바르게 살고 있는가?"**

## 5. 주요 논의 내용

| 논의 주제 | 논의 내용 |
|---|---|
| **사용자의 진짜 문제** | 이희망은 단순히 피곤한 사람이 아니라, 지금의 삶이 하나님께 가까운 방향인지 확신하지 못해 불안한 사람으로 정의함 |
| **필요한 메시지 톤** | 훈계하거나 정죄하는 말보다, 현실의 무게를 이해하고 붙들어주는 공감형 메시지가 필요하다고 판단함 |
| **서비스 역할** | 직장을 떠나야 하나님께 가까워지는 것이 아니라, 직장 안에서도 하나님을 향한 삶을 선택할 수 있도록 돕는 역할을 해야 함 |
| **실행 방식** | 긴 설교형 콘텐츠보다 하루 5분 안에 읽고 실천할 수 있는 짧은 메시지와 행동 제안이 적합하다고 정리함 |

## 7. 서비스가 줘야 하는 메시지

| 메시지 방향 | 설명 |
|---|---|
| **공감** | 50대 직장인의 책임과 피로를 먼저 인정한다 |
| **관점 전환** | 직장도 하나님과 멀어지는 장소가 아니라 신앙을 실천하는 자리로 재해석한다 |
| **작은 실천** | 거창한 결단보다 하루 5분 안에 가능한 행동을 제안한다 |
| **평안 회복** | 천국은 미래의 장소만이 아니라 오늘의 선택 속에서도 시작된다는 메시지를 전한다 |

> **"천국을 향한 삶은 현실을 버리는 것이 아니라, 현실 속 선택을 하나님께 가까이 맞춰가는 것입니다."**

## 9. 콘텐츠 구조 제안

| 단계 | 목적 | 설명 |
|---|---|---|

| **관점 전환** | 삶을 새롭게 해석한다 | 직장과 일상도 하나님 앞에서 의미 있는 자리로 바라보게 한다 |
| **작은 실천** | 오늘의 행동으로 연결한다 | 5분 안에 가능한 구체적 행동을 제안한다 |

| **작은 실천** | 회의 전에 10초만 멈추고, 상대를 이기기보다 이해하게 해달라고 마음속으로 기도해보세요. |

## 10. 콘텐츠 예시

### 오늘의 메시지 예시

> **이희망 부장님, 오늘도 회의와 보고와 책임 속에서 마음이 바쁘실 수 있습니다.  
하지만 하나님은 당신이 회사 밖으로 나와야만 만날 수 있는 분이 아닙니다.  
오늘의 책상, 오늘의 말 한마디, 오늘의 선택 속에서도 하나님께 가까이 갈 수 있습니다.**

### 오늘의 5분 실천

| 상황 | 실천 | 의도 |
|---|---|---|
| 출근 직후 | "오늘 내 말이 누군가를 살리게 해주세요"라고 짧게 기도하기 | 하루의 방향 설정 |
| 회의 전 | 상대를 이기려 하기보다 이해하려는 마음 갖기 | 관계의 태도 전환 |
| 업무 중 실수했을 때 | 자책보다 정직하게 인정하고 바로잡기 | 양심과 평안 지키기 |
| 퇴근 전 | 오늘 양심을 지킨 순간 하나를 떠올리기 | 하루 의미 회복 |
| 잠들기 전 | 감사 1가지, 맡길 일 1가지 적기 | 마음 정리와 평안 |

## 11. AI 코칭 응답 구조 초안

| 단계 | 챗봇 역할 | 예시 |
|---|---|---|
| **1단계** | 감정 확인 | "오늘 많이 무거우셨겠어요." |
| **2단계** | 상황 의미화 | "그 마음은 단순한 피로가 아니라 방향을 점검하고 싶은 마음일 수 있어요." |
| **3단계** | 관점 전환 | "직장도 하나님과 멀어지는 장소가 아니라, 믿음을 선택하는 자리일 수 있습니다." |
| **4단계** | 작은 실천 제안 | "지금 10초만 눈을 감고 오늘 지키고 싶은 한 가지를 떠올려보세요." |
| **5단계** | 짧은 마무리 | "오늘도 충분히 의미 있는 하루를 시작하고 계십니다." |

## 14. 회의 마무리

> **"이 서비스는 성공한 사람을 더 성공하게 만드는 게 아니라,  
지친 사람이 자기 삶을 다시 하나님 앞에서 바라보게 만드는 서비스였으면 좋겠습니다."**

### 최종 정리

| 구분 | 내용 |
|---|---|
| **대상 사용자** | 신앙과 직장 현실 사이에서 갈등하는 50대 직장인 |
| **핵심 문제** | 삶의 방향이 하나님께 가까운지 확신하지 못하는 불안 |
| **서비스 역할** | 정죄하지 않고 공감하며, 하루 속 신앙 실천을 돕는 코칭 제공 |
| **핵심 구조** | 공감 → 관점 전환 → 작은 실천 |
| **서비스 이름** | 오늘의 천국 연습 |
| **기대 효과** | 직장과 신앙의 분열감을 줄이고, 하루의 평안과 방향성을 회복하게 함 |
